# Projet MediaTek86
Application C# pour la gestion des absences de MediaTek86.

#Installation
Installe l'app avec MediaTek86_Setup.exe.

Pour la base de données, lance le fichier mediatek86_install.sql dans MySQL.

#Contenu
Code source : Dossier avec le projet C#.

Installation : Les fichiers MediaTek86_Setup.msi et .exe.

Base de données : Le script mediatek86_install.sql qui crée tout proprement.
